hyperion::mpl::Value#

group value

Hyperion provides mpl::Value as a metaprogramming type for storing, communicating, working with, and calculating compile-time values.

Example#

#include <hyperion/mpl/value.h>

using namespace hyperion::mpl;

constexpr auto val1 = Value<4>{};
constexpr auto val2 = Value<2>{};

constexpr auto meaning_of_life = (val1 * 10_value) + val2;

static_assert(meaning_of_life == 42);

template<typename TType> concept ValueType   = requires { TType::value; } && detail::HasStaticConstexprValue<TType>

Concept specifying the requirements for a metaprogramming value type.

A metaprogramming value type is any type that has a static constexpr member variable named value.

Requirements#

  • TType must have a static constexpr member variable named value.
